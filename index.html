жһ„е»әжҢҮд»ӨпјҢз”ЁдәҺжҢҮе®ҡеҹәзЎҖimageгҖӮиҜҘжҢҮд»Өеҝ…йЎ»жҢҮе®ҡеңЁDockerfileе…¶е®ғжҢҮд»Өзҡ„еүҚйқўпјҢеҗҺйқўзҡ„DockerfileжҢҮд»ӨйғҪдҫқиө–дәҺиҜҘжҢҮд»ӨжҢҮе®ҡзҡ„imageгҖӮFROMжҢҮд»ӨжҢҮе®ҡзҡ„еҹәзЎҖimageеҸҜд»ҘжҳҜе®ҳж–№docker hubд»“еә“дёӯзҡ„пјҢд№ҹеҸҜд»ҘжҳҜдҪҚдәҺжң¬ең°д»“еә“дёӯзҡ„гҖӮ

жҢҮд»Өж јејҸпјҡFROM <image>[:<tag>]

жһ„е»әжҢҮд»ӨпјҢеҸҜд»ҘиҝҗиЎҢд»»дҪ•иў«еҹәзЎҖimageж”ҜжҢҒзҡ„е‘Ҫд»ӨгҖӮ

жҢҮд»Өж јејҸпјҡRUN <command> жҲ– RUN ["executable","param 1","param n"]
дёҖиҲ¬з”ЁдәҺиҪҜд»¶е®үиЈ…

CMD

и®ҫзҪ®жҢҮд»ӨпјҢз”ЁдәҺcontainerеҗҜеҠЁж—¶жҢҮе®ҡзҡ„ж“ҚдҪңгҖӮиҜҘж“ҚдҪңеҸҜд»ҘжҳҜжү§иЎҢиҮӘе®ҡд№үи„ҡжң¬пјҢд№ҹеҸҜд»ҘжҳҜжү§иЎҢзі»з»ҹе‘Ҫд»ӨгҖӮиҜҘжҢҮд»ӨеҸӘиғҪеңЁж–Үд»¶дёӯеӯҳеңЁдёҖж¬ЎпјҢеҰӮжһңжңүеӨҡдёӘпјҢеҲҷеҸӘжү§иЎҢжңҖеҗҺдёҖжқЎгҖӮ

жҢҮд»Өж јејҸпјҡCMD <executable> <param 1> <param n> жҲ– CMD ["executable", "param 1", "param n"]

ENTRYPOINT

и®ҫзҪ®жҢҮд»ӨпјҢжҢҮе®ҡе®№еҷЁеҗҜеҠЁж—¶жү§иЎҢзҡ„е‘Ҫд»ӨпјҢеҸҜд»ҘеӨҡж¬Ўи®ҫзҪ®пјҢдҪҶжҳҜеҸӘжңүжңҖеҗҺдёҖдёӘжңүж•ҲгҖӮ

жҢҮд»Өж јејҸпјҡENTRYPOINT ["executable", "param 1", "param n"] жҲ– ENTRYPOINT <executable> <param 1> <param n>
CMDжҢҮд»Өе’ҢENTRYPOINTдјҡдә’зӣёиҰҶзӣ–еҸӘжңүжңҖеҗҺдёҖдёӘCMDжҲ–иҖ…ENTRYPOINTжңүж•ҲгҖӮ

е’ҢCMDжҢҮд»Өй…ҚеҗҲдҪҝз”ЁжқҘжҢҮе®ҡENTRYPOINTзҡ„й»ҳи®ӨеҸӮж•°пјҢиҝҷж—¶CMDжҢҮд»ӨдёҚжҳҜдёҖдёӘе®Ңж•ҙзҡ„еҸҜжү§иЎҢе‘Ҫд»ӨпјҢд»…д»…жҳҜеҸӮж•°йғЁеҲҶгҖӮENTRYPOINTжҢҮд»ӨеҸӘиғҪдҪҝз”ЁJSONж–№ејҸжҢҮе®ҡжү§иЎҢе‘Ҫд»ӨпјҢиҖҢдёҚиғҪжҢҮе®ҡеҸӮж•°гҖӮ

USER

и®ҫзҪ®жҢҮд»ӨпјҢи®ҫзҪ®еҗҜеҠЁе®№еҷЁзҡ„з”ЁжҲ·пјҢй»ҳи®ӨжҳҜrootз”ЁжҲ·

жҢҮд»Өж јејҸпјҡUSER <user name>

EXPOSE

и®ҫзҪ®жҢҮд»ӨпјҢиҜҘжҢҮд»Өдјҡе°Ҷе®№еҷЁдёӯзҡ„з«ҜеҸЈжҳ е°„жҲҗе®ҝдё»жңәеҷЁдёӯзҡ„жҹҗдёӘз«ҜеҸЈгҖӮйңҖиҰҒи®ҝй—®е®№еҷЁзҡ„ж—¶еҖҷпјҢеҸҜд»ҘдёҚжҳҜз”Ёе®№еҷЁзҡ„IPең°еқҖиҖҢжҳҜдҪҝз”Ёе®ҝдё»жңәеҷЁзҡ„IPең°еқҖе’Ңжҳ е°„еҗҺзҡ„з«ҜеҸЈгҖӮйҰ–е…ҲеңЁDockerfileдҪҝз”ЁEXPOSEи®ҫзҪ®йңҖиҰҒжҳ е°„зҡ„е®№еҷЁз«ҜеҸЈпјҢз„¶еҗҺеңЁиҝҗиЎҢе®№еҷЁзҡ„ж—¶еҖҷжҢҮе®ҡ-pйҖүйЎ№еҠ дёҠEXPOSEи®ҫзҪ®зҡ„з«ҜеҸЈпјҢиҝҷж ·EXPOSEи®ҫзҪ®зҡ„з«ҜеҸЈеҸ·дјҡиў«йҡҸжңәжҳ е°„жҲҗе®ҝдё»жңәеҷЁдёӯзҡ„дёҖдёӘз«ҜеҸЈеҸ·гҖӮд№ҹеҸҜд»ҘжҢҮе®ҡйңҖиҰҒжҳ е°„еҲ°е®ҝдё»жңәеҷЁзҡ„йӮЈдёӘз«ҜеҸЈпјҢиҝҷж—¶иҰҒзЎ®дҝқе®ҝдё»жңәеҷЁдёҠзҡ„з«ҜеҸЈеҸ·жІЎжңүиў«дҪҝз”ЁгҖӮEXPOSEжҢҮд»ӨеҸҜд»ҘдёҖж¬Ўи®ҫзҪ®еӨҡдёӘз«ҜеҸЈеҸ·пјҢзӣёеә”зҡ„иҝҗиЎҢе®№еҷЁзҡ„ж—¶еҖҷпјҢеҸҜд»Ҙй…ҚеҘ—зҡ„еӨҡж¬ЎдҪҝз”Ё-pйҖүйЎ№гҖӮ

жҢҮд»Өж јејҸпјҡEXPOSE <port> [<port>...]

ENV

и®ҫзҪ®дәҶеҗҺпјҢеҗҺз»ӯзҡ„RUNе‘Ҫд»ӨйғҪеҸҜд»ҘдҪҝз”ЁпјҢcontainerеҗҜеҠЁеҗҺпјҢеҸҜд»ҘйҖҡиҝҮdocker inspectжҹҘзңӢиҝҷдёӘзҺҜеўғеҸҳйҮҸпјҢд№ҹеҸҜд»ҘйҖҡиҝҮеңЁdocker run вҖ“env key=valueж—¶и®ҫзҪ®жҲ–дҝ®ж”№зҺҜеўғеҸҳйҮҸгҖӮ
