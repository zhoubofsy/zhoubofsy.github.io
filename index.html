еҲқе°қGlusterFS

еҸ‘иЎЁдәҺ 2017-08-03  |  2017-08-03   |   еҲҶзұ»дәҺ storage

жҰӮиҝ°

GlusterFS (Gluster File System) жҳҜдёҖдёӘејҖжәҗзҡ„еҲҶеёғејҸж–Үд»¶зі»з»ҹпјҢдё»иҰҒз”ұ Z RESEARCH е…¬еҸёиҙҹиҙЈејҖеҸ‘гҖӮGlusterFS жҳҜ Scale-Out еӯҳеӮЁи§ЈеҶіж–№жЎҲ Gluster зҡ„ж ёеҝғпјҢе…·жңүејәеӨ§зҡ„жЁӘеҗ‘жү©еұ•иғҪеҠӣпјҢйҖҡиҝҮжү©еұ•иғҪеӨҹж”ҜжҢҒж•°PBеӯҳеӮЁе®№йҮҸе’ҢеӨ„зҗҶж•°еҚғе®ўжҲ·з«ҜгҖӮGlusterFS еҖҹеҠ© TCP/IP жҲ– InfiniBand RDMA зҪ‘з»ңе°Ҷзү©зҗҶеҲҶеёғзҡ„еӯҳеӮЁиө„жәҗиҒҡйӣҶеңЁдёҖиө·пјҢдҪҝз”ЁеҚ•дёҖе…ЁеұҖе‘ҪеҗҚз©әй—ҙжқҘз®ЎзҗҶж•°жҚ®гҖӮGlusterFS еҹәдәҺеҸҜе ҶеҸ зҡ„з”ЁжҲ·з©әй—ҙи®ҫи®ЎпјҢеҸҜдёәеҗ„з§ҚдёҚеҗҢзҡ„ж•°жҚ®иҙҹиҪҪжҸҗдҫӣдјҳејӮзҡ„жҖ§иғҪгҖӮ

GlusterFS жҖ»дҪ“жһ¶жһ„дёҺз»„жҲҗпјҢе®ғдё»иҰҒз”ұеӯҳеӮЁжңҚеҠЎеҷЁпјҲBrick ServerпјүгҖҒе®ўжҲ·з«Ҝд»ҘеҸҠ NFS/Samba еӯҳеӮЁзҪ‘е…із»„жҲҗгҖӮдёҚйҡҫеҸ‘зҺ°пјҢGlusterFS жһ¶жһ„дёӯжІЎжңүе…ғж•°жҚ®жңҚеҠЎеҷЁз»„д»¶пјҢиҝҷжҳҜе…¶жңҖеӨ§зҡ„и®ҫи®ЎиҝҷзӮ№пјҢеҜ№дәҺжҸҗеҚҮж•ҙдёӘзі»з»ҹзҡ„жҖ§иғҪгҖҒеҸҜйқ жҖ§е’ҢзЁіе®ҡжҖ§йғҪжңүзқҖеҶіе®ҡжҖ§зҡ„ж„Ҹд№үгҖӮ

  • GlusterFS ж”ҜжҢҒ TCP/IP е’Ң InfiniBand RDMA й«ҳйҖҹзҪ‘з»ңдә’иҒ”гҖӮ
  • е®ўжҲ·з«ҜеҸҜйҖҡиҝҮеҺҹз”ҹ GlusterFS еҚҸи®®и®ҝй—®ж•°жҚ®пјҢе…¶д»–жІЎжңүиҝҗиЎҢ GlusterFS е®ўжҲ·з«Ҝзҡ„з»Ҳз«ҜеҸҜйҖҡиҝҮ NFS/CIFS ж ҮеҮҶеҚҸи®®йҖҡиҝҮеӯҳеӮЁзҪ‘е…іи®ҝй—®ж•°жҚ®пјҲеӯҳеӮЁзҪ‘е…іжҸҗдҫӣеј№жҖ§еҚ·з®ЎзҗҶе’Ңи®ҝй—®д»ЈзҗҶеҠҹиғҪпјүгҖӮ
  • еӯҳеӮЁжңҚеҠЎеҷЁдё»иҰҒжҸҗдҫӣеҹәжң¬зҡ„ж•°жҚ®еӯҳеӮЁеҠҹиғҪпјҢе®ўжҲ·з«ҜејҘиЎҘдәҶжІЎжңүе…ғж•°жҚ®жңҚеҠЎеҷЁзҡ„й—®йўҳпјҢжүҝжӢ…дәҶжӣҙеӨҡзҡ„еҠҹиғҪпјҢеҢ…жӢ¬ж•°жҚ®еҚ·з®ЎзҗҶгҖҒI/O и°ғеәҰгҖҒж–Үд»¶е®ҡдҪҚгҖҒж•°жҚ®зј“еӯҳзӯүеҠҹиғҪпјҢеҲ©з”Ё FUSEпјҲFile system in User SpaceпјүжЁЎеқ—е°Ҷ GlusterFS жҢӮиҪҪеҲ°жң¬ең°ж–Үд»¶зі»з»ҹд№ӢдёҠпјҢе®һзҺ° POSIX е…је®№зҡ„ж–№ејҸжқҘи®ҝй—®зі»з»ҹж•°жҚ®гҖӮ

еҚ·зұ»еһӢ

DistributeеҚ·

еҲҶеёғејҸеҚ·пјҢеҹәдәҺ Hash з®—жі•е°Ҷж–Үд»¶еҲҶеёғеҲ°жүҖжңү brick serverпјҢеҸӘжҳҜжү©еӨ§дәҶзЈҒзӣҳз©әй—ҙпјҢдёҚе…·еӨҮе®№й”ҷиғҪеҠӣгҖӮз”ұдәҺdistribute volume дҪҝз”Ёжң¬ең°ж–Үд»¶зі»з»ҹпјҢеӣ жӯӨеӯҳеҸ–ж•ҲзҺҮе№¶жІЎжңүжҸҗй«ҳпјҢзӣёеҸҚдјҡеӣ дёәзҪ‘з»ңйҖҡдҝЎзҡ„еҺҹеӣ дҪҝз”Ёж•ҲзҺҮжңүжүҖйҷҚдҪҺпјҢеҸҰеӨ–жң¬ең°еӯҳеӮЁи®ҫеӨҮзҡ„е®№йҮҸжңүйҷҗеҲ¶пјҢеӣ жӯӨж”ҜжҢҒи¶…еӨ§еһӢж–Үд»¶дјҡжңүдёҖе®ҡйҡҫеәҰгҖӮ

StripeеҚ·

жқЎеёҰеҚ·пјҢзұ»дјј RAID0пјҢж–Үд»¶еҲҶжҲҗж•°жҚ®еқ—д»Ҙ Round Robin ж–№ејҸеҲҶеёғеҲ° brick server дёҠпјҢе№¶еҸ‘зІ’еәҰжҳҜж•°жҚ®еқ—пјҢж”ҜжҢҒи¶…еӨ§ж–Үд»¶пјҢеӨ§ж–Үд»¶зҡ„иҜ»еҶҷжҖ§иғҪй«ҳгҖӮ

ReplicaеҚ·

еӨҚеҲ¶еҚ·пјҢж–Үд»¶еҗҢжӯҘеӨҚеҲ¶еҲ°еӨҡдёӘ brick дёҠпјҢж–Үд»¶зә§ RAID1пјҢе…·жңүе®№й”ҷиғҪеҠӣпјҢеҶҷжҖ§иғҪдёӢйҷҚпјҢиҜ»жҖ§иғҪжҸҗеҚҮгҖӮReplicated жЁЎејҸпјҢд№ҹз§°дҪң AFRпјҲAuto File ReplicationпјүпјҢзӣёеҪ“дәҺ RAID1пјҢеҚіеҗҢдёҖж–Үд»¶еңЁеӨҡдёӘй•ңеғҸеӯҳеӮЁиҠӮзӮ№дёҠдҝқеӯҳеӨҡд»ҪпјҢжҜҸдёӘ replicated еӯҗиҠӮзӮ№жңүзқҖзӣёеҗҢзҡ„зӣ®еҪ•з»“жһ„е’Ңж–Үд»¶пјҢreplica volume д№ҹжҳҜеңЁе®№еҷЁеӯҳеӮЁдёӯиҫғдёәжҺЁеҙҮзҡ„дёҖз§ҚгҖӮ

Distribute StripeеҚ·

еҲҶеёғејҸжқЎеёҰеҚ·пјҢBrick server ж•°йҮҸжҳҜжқЎеёҰж•°зҡ„еҖҚж•°пјҢе…је…· distribute е’Ң stripe еҚ·зҡ„зү№зӮ№гҖӮеҲҶеёғејҸзҡ„жқЎеёҰеҚ·пјҢvolume дёӯ brick жүҖеҢ…еҗ«зҡ„еӯҳеӮЁжңҚеҠЎеҷЁж•°еҝ…йЎ»жҳҜ stripe зҡ„еҖҚж•°(>=2еҖҚ)пјҢе…јйЎҫеҲҶеёғејҸе’ҢжқЎеёҰејҸзҡ„еҠҹиғҪгҖӮжҜҸдёӘж–Үд»¶еҲҶеёғеңЁеӣӣеҸ°е…ұдә«жңҚеҠЎеҷЁдёҠпјҢйҖҡеёёз”ЁдәҺеӨ§ж–Үд»¶и®ҝй—®еӨ„зҗҶпјҢжңҖе°‘йңҖиҰҒ 4 еҸ°жңҚеҠЎеҷЁжүҚиғҪеҲӣе»әеҲҶеёғжқЎеёҰеҚ·гҖӮ

еҲҶеёғејҸеӨҚеҲ¶еҚ·пјҢBrick server ж•°йҮҸжҳҜй•ңеғҸж•°зҡ„еҖҚж•°пјҢе…је…· distribute е’Ң replica еҚ·зҡ„зү№зӮ№,еҸҜд»ҘеңЁ 2 дёӘжҲ–еӨҡдёӘиҠӮзӮ№д№Ӣй—ҙеӨҚеҲ¶ж•°жҚ®гҖӮеҲҶеёғејҸзҡ„еӨҚеҲ¶еҚ·пјҢvolume дёӯ brick жүҖеҢ…еҗ«зҡ„еӯҳеӮЁжңҚеҠЎеҷЁж•°еҝ…йЎ»жҳҜ replica зҡ„еҖҚж•°(>=2еҖҚ)пјҢе…јйЎҫеҲҶеёғејҸе’ҢеӨҚеҲ¶ејҸзҡ„еҠҹиғҪгҖӮ
