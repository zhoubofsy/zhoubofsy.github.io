RPI WIFI用户名密码配置

发表于 2017-01-02 | 分类于 rpi

介绍

Raspberrypi3自带wifi模块，用户可以通过配置用户名密码就可以建立wifi连接。

版本

  • 硬件，RaspberryPi3
  • OS，raspbian jessie

配置

使用wpa_passphrase <SSID> <用户名> <密码>生成wifi连接需要的用户信息
然后，将wpa_passphrase生成的信息，追加到/etc/wpa_supplicant/wpa_supplicant.conf中
最后重启系统。
