Docker网络管理——PIPEWORK

发表于 2017-05-08  |  2017-05-09   |   分类于 container

PIPEWORK 是一个简单易用的Docker容器网络配置工具，由Docker的工程师Jérôme Petazzoni开发。通过使用ip、brctl、ovs-vsctl等命令来为Docker容器配置自定义的网桥、网卡、路由等。

  • 使用新建的br0网桥替代默认的docker0网桥
  • 容器网络类型选择none
